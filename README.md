# Parabank Testing Project

## Overview:
This project focuses on manual testing of the Parabank web application. The objective is to ensure 
that all core banking functionalities work correctly and meet the expected requirements.

## Objectives
* Validate end-to-end user workflows
* Identify functional and UI defects
* Ensure data validation and security checks
* Improve overall application quality

## Modules Covered
* Registration
* Login
* Open New Account
* Accounts Overview
* Transfer Funds
* Bill Pay
* Find Transactions
* Request Loan
* Logout

## Testing Types Performed
* Functional Testing
* UI Testing
* Regression Testing
* Smoke Testing
* Sanity Testing
* System Testing
* Security Testing (SQL Injection, XSS)
* Performance Testing

## Test Case Design
* Designed detailed test cases covering:
  * Positive scenarios
  * Negative scenarios
  * Boundary value analysis
*Included:
  * Preconditions
  * Test Scenarios
  * Test Steps
  * Test Data
  * Expected Results
  * Actual Results
  * Status (Pass/Fail)
